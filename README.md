## 🚀 Local Deployment
```bash
git clone [https://github.com/bartosz-wawrzyk/Portfolio.git](https://github.com/bartosz-wawrzyk/Portfolio.git)
cd Portfolio
npm install
npm run dev
```
Open http://localhost:5173 in your browser
